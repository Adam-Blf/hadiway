# HadiWay

![Status](https://img.shields.io/badge/status-ready-brightgreen)

## Description

HadiWay est une Progressive Web App (PWA) révolutionnaire conçue pour simplifier la mobilité des Personnes à Mobilité Réduite (PMR). L'application offre un calcul d'itinéraire accessible et la possibilité de signaler collaborativement des obstacles, avec une interface Premium "Apple-like".

## Ce Dépôt Contient

- Code source **Next.js PWA** (Dossier `src`) : Frontend réactif avec carte `react-leaflet`.
- **HadiWay_La_Bible_Des_Debutants_90_Pages.pdf** : Un guide ultra-détaillé de plus de 90 pages expliquant comment installer et développer ce projet de A à Z (Garanti 20/20).
- Scripts d'icônes PWA.

## Features

- [x] Progressive Web App (PWA) - Installable sur mobile
- [x] Carte Interactive (React-Leaflet) sans clé API
- [x] Interface Apple-like (Tailwind CSS, Glassmorphism, Animations)
- [x] Routing simulé et POIs accessibles
- [x] Guide PDF de 90 pages pour débutant généré en Python.

## Installation

Assurez-vous d'avoir Node.js installé.

```bash
git clone https://github.com/Adam-Blf/hadiway.git
cd hadiway
npm install --legacy-peer-deps
npm run dev
```

Ouvrez `http://localhost:3000` dans votre navigateur.
